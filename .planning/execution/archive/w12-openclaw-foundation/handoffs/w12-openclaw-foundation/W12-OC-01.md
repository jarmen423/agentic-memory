## What Changed

- Added multi-key auth support via `AM_SERVER_API_KEYS` with backward compatibility for `AM_SERVER_API_KEY`.
- Added a shared machine-readable error envelope with request id in both body and headers.
- Added in-process Prometheus-style request/error metrics and an authenticated `/metrics` route.
- Expanded backend tests to cover auth, metrics, validation, and runtime-failure contracts.
