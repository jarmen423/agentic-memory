    ```python
    class EmbeddingService:
        """Provider-dispatching embedding service.

        Wraps OpenAI, Gemini, and Nemotron behind a single .embed() interface.
        Each ingestion module instantiates its own EmbeddingService with the right provider.
        """

        PROVIDERS = {
            "openai": {"model": "text-embedding-3-large", "dimensions": 3072},
            "gemini": {"model": "gemini-embedding-001", "dimensions": 768},
            "nemotron": {"model": "nvidia/nv-embedqa-e5-v5", "dimensions": 4096},
        }
    ```

    **Constructor `__init__(self, provider: str, api_key: str, base_url: str | None = None, output_dimensions: int | None = None)`:**
    - Validate `provider` is in PROVIDERS, raise `ValueError` if not
    - Set `self.provider`, `self.model`, `self.dimensions` (from output_dimensions or PROVIDERS default)
    - For "gemini": `self._client = genai.Client(api_key=api_key)`
    - For "openai": `self._client = OpenAI(api_key=api_key)`
    - For "nemotron": `self._client = OpenAI(api_key=api_key, base_url=base_url or "https://integrate.api.nvidia.com/v1")`

    **Method `embed(self, text: str) -> list[float]`:**
    - For "gemini": call `self._client.models.embed_content(model=self.model, contents=text, config={"output_dimensionality": self.dimensions})` and return `result.embeddings[0].values`
    - CRITICAL: ALWAYS pass `output_dimensionality` to Gemini — default is 3072 but we need 768 for web/chat indexes. This is Pitfall 2 from RESEARCH.md.
    - For "openai"/"nemotron": call `self._client.embeddings.create(model=self.model, input=text, dimensions=self.dimensions)` and return `response.data[0].embedding`

    **Method `embed_batch(self, texts: list[str]) -> list[list[float]]`:**
    - For "gemini": loop over texts calling embed() individually (genai SDK does not support batch natively in a way that returns per-text vectors easily)
    - For "openai"/"nemotron": single API call with `input=texts`, return `[d.embedding for d in response.data]`

    **Property `model_info(self) -> dict[str, Any]`:**
    - Returns `{"provider": self.provider, "model": self.model, "dimensions": self.dimensions}`

    Module logger: `logger = logging.getLogger(__name__)`
    Imports: `logging`, `typing.Any`, `google.genai` (import at top level), `openai.OpenAI` (import at top level)

    **Create `src/codememory/core/entity_extraction.py`:**

    Module-level constant for the extraction prompt:
    ```python
    ENTITY_EXTRACTION_PROMPT = """\
    Extract named entities from the following text.
    Return a JSON object with key "entities" containing a list of objects,
    each with "name" (string) and "type" (one of: {allowed_types}).
    Only extract entities clearly present in the text.
    Do not invent entities. If no entities found, return {"entities": []}."""
    ```

    **Class `EntityExtractionService`:**
    - `__init__(self, api_key: str, model: str = "llama-3.3-70b-versatile", allowed_types: list[str] | None = None) -> None`
    - Creates `self._client = Groq(api_key=api_key)`
    - Sets `self.model = model`
    - Sets `self.allowed_types = allowed_types or ["project", "person", "business", "technology", "concept"]`

    **Method `extract(self, document_text: str) -> list[dict[str, str]]`:**
    - Formats prompt with `", ".join(self.allowed_types)`
    - Truncates document_text to first 8000 characters (budget guard per RESEARCH.md)
    - Calls `self._client.chat.completions.create(model=self.model, messages=[{"role": "system", "content": prompt}, {"role": "user", "content": truncated_text}], response_format={"type": "json_object"}, temperature=0.0)`
    - Parses response JSON with `json.loads(response.choices[0].message.content)`
    - Gets entities from `data.get("entities")`. If None (wrong key, per Pitfall 4), scan `data.values()` for first value that is a list and use that.
    - Filters entities: keep only those where `entity["type"]` is in `self.allowed_types`
    - Returns `list[dict[str, str]]` — each dict has `"name"` and `"type"` keys

    **Module-level function `build_embed_text(chunk_text: str, entities: list[dict[str, str]]) -> str`:**
    - If entities is empty, return chunk_text unchanged
    - Otherwise: `entity_str = ", ".join(f"{e['name']} ({e['type']})" for e in entities)` then return `f"Context: {entity_str}\n\n{chunk_text}"`
    - This is the entity-enriched embedding text per CONTEXT.md locked decision.

    Module logger: `logger = logging.getLogger(__name__)`
    Imports: `json`, `logging`, `groq.Groq`, `typing.Any`
