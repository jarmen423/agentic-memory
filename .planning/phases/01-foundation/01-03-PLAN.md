    ```python
    import abc
    from typing import Any
    from codememory.core.connection import ConnectionManager
    from codememory.core.registry import SOURCE_REGISTRY

    class BaseIngestionPipeline(abc.ABC):
        """Base class for all memory ingestion pipelines.

        Subclasses MUST declare:
            DOMAIN_LABEL: str  — "Code", "Research", or "Conversation"

        Subclasses MUST implement:
            ingest(source) — domain-specific ingestion logic
        """

        DOMAIN_LABEL: str  # Subclass MUST set this class variable

        def __init__(self, connection_manager: ConnectionManager) -> None:
            self._conn = connection_manager

        @abc.abstractmethod
        def ingest(self, source: Any) -> dict[str, Any]:
            """Ingest a source document. Returns ingestion summary dict."""

        def node_labels(self, source_key: str) -> list[str]:
            """Get node labels from source registry, with fallback."""
            return SOURCE_REGISTRY.get(source_key, ["Memory", self.DOMAIN_LABEL])
    ```

    The class does NOT include parsing, chunking, or embedding logic — those stay in subclasses per CONTEXT.md decision. The base class provides:
    1. Connection access via `self._conn`
    2. Label resolution via `node_labels()`
    3. Abstract contract via `ingest()`

    **Create `src/codememory/core/graph_writer.py`:**

    ```python
    class GraphWriter:
        """Shared Neo4j write patterns for Memory and Entity nodes.

        All writes use MERGE for idempotency (re-ingest same content = same node).
        Uses composite key (source_key, content_hash) per CONTEXT.md anti-pattern guidance.
        """

        def __init__(self, connection_manager: ConnectionManager) -> None:
            self._conn = connection_manager
    ```

    **Method `write_memory_node(self, labels: list[str], properties: dict[str, Any]) -> None`:**
    - `labels` is a list like `["Memory", "Code", "Chunk"]` — joined as `:Memory:Code:Chunk` in Cypher
    - `properties` MUST contain ALL required metadata fields:
      - `source_key` (str), `content_hash` (str), `session_id` (str), `source_type` (str),
      - `ingested_at` (str — ISO datetime), `ingestion_mode` (str — "active" or "passive"),
      - `embedding_model` (str), `project_id` (str), `entities` (list[str] — denormalized names),
      - `entity_types` (list[str] — denormalized types), `embedding` (list[float]), `text` (str)
    - Cypher template:
      ```
      MERGE (m:{labels_str} {source_key: $source_key, content_hash: $content_hash})
      ON CREATE SET m += $props
      ON MATCH SET m.ingested_at = $ingested_at
      ```
    - Build `labels_str` by joining labels with `:` prefix: `":".join(labels)` prefixed with `:`
    - Pass all properties as parameters (never interpolate values into Cypher strings)
    - Use `self._conn.session()` context manager to execute

    **Method `upsert_entity(self, name: str, entity_type: str) -> None`:**
    - MERGE on composite key `(name, type)` with labels `:Entity:{entity_type.capitalize()}`
    - Cypher: `MERGE (e:Entity:{type_label} {name: $name, type: $type})`
    - The entity type label is the capitalized entity_type: e.g., "technology" -> `:Entity:Technology`

    **Method `write_relationship(self, source_key: str, content_hash: str, entity_name: str, entity_type: str, rel_type: str = "ABOUT") -> None`:**
    - MATCH the Memory node by (source_key, content_hash)
    - MATCH the Entity node by (name, type)
    - MERGE the relationship: `(m)-[:{rel_type}]->(e)`
    - `rel_type` defaults to "ABOUT" but can be "MENTIONS" or "BELONGS_TO"

    Module logger: `logger = logging.getLogger(__name__)`

    ```python
    import logging
    from typing import Any
    from codememory.core.embedding import EmbeddingService

    logger = logging.getLogger(__name__)

    # Maps domain to its expected vector index dimensions
    LABEL_DIMENSION_MAP: dict[str, int] = {
        "code": 3072,      # OpenAI text-embedding-3-large
        "web": 768,         # Gemini gemini-embedding-001 @ 768d
        "chat": 768,        # Gemini gemini-embedding-001 @ 768d
    }
    ```

    **Function `validate_embedding_config(config: dict[str, Any]) -> None`:**
    - If `"modules"` not in config, return immediately (nothing to validate)
    - For each module_name, module_cfg in config["modules"].items():
      - Get `provider = module_cfg.get("embedding_provider", "")`
      - If provider not in EmbeddingService.PROVIDERS, raise `ValueError(f"Unknown embedding provider '{provider}' for module '{module_name}'. Valid providers: {list(EmbeddingService.PROVIDERS.keys())}")`
      - Get `configured_dims = module_cfg.get("embedding_dimensions")`
      - Get `provider_default_dims = EmbeddingService.PROVIDERS[provider]["dimensions"]`
      - Get `expected_index_dims = LABEL_DIMENSION_MAP.get(module_name)`
      - If configured_dims is not None and expected_index_dims is not None:
        - For Gemini provider: allow any configured_dims (Gemini supports output_dimensionality override) — but log a warning if configured_dims != expected_index_dims
        - For OpenAI/Nemotron: if configured_dims != provider_default_dims, raise `ValueError(f"Module '{module_name}' configured with {configured_dims}d but provider '{provider}' outputs {provider_default_dims}d. Dimension mismatch will cause Neo4j vector index errors.")`
    - Log `logger.info(f"Embedding config validated for {len(config['modules'])} modules")`
