### Task 2: Implement config.py — TOML loader with defaults

**File:** `packages/am-proxy/src/am_proxy/config.py`

**Action:** Create `config.py` with a `ProxyConfig` dataclass and `load_config()` function. Full specification:

```python
"""Configuration loading for am-proxy.

Config file location: ~/.config/am-proxy/config.toml
Missing file is not an error — all fields have defaults.
CLI flags override config file values when passed.
"""

from __future__ import annotations

import dataclasses
from pathlib import Path
from typing import Any

try:
    import tomllib
except ImportError:
    import tomli as tomllib  # type: ignore[no-redef]

# Default config file path
DEFAULT_CONFIG_PATH = Path.home() / ".config" / "am-proxy" / "config.toml"


@dataclasses.dataclass
class ProxyConfig:
    """Runtime configuration for am-proxy.

    All fields have defaults so the proxy works with zero configuration.
    """

    endpoint: str = "http://localhost:8000"
    api_key: str = ""
    default_project_id: str = "default"
    timeout_seconds: float = 5.0
    buffer_ttl_seconds: float = 300.0
    debug: bool = False
```

The `load_config()` function must:
1. Accept an optional `config_path: Path | None = None` parameter (uses `DEFAULT_CONFIG_PATH` if None)
2. If the file does not exist, return `ProxyConfig()` with all defaults — do NOT raise
3. If the file exists, open it in binary mode with `tomllib.load()`, extract the `[am_proxy]` section if present, and build `ProxyConfig` from it
4. Support all `ProxyConfig` fields as TOML keys under `[am_proxy]` section (snake_case matches)
5. Unknown keys in the TOML are silently ignored (use `.get()` per field)
6. Signature: `def load_config(config_path: Path | None = None) -> ProxyConfig:`

Full implementation:

```python
def load_config(config_path: Path | None = None) -> ProxyConfig:
    """Load ProxyConfig from TOML file, returning defaults if file absent.

    Args:
        config_path: Path to TOML config file. Defaults to ~/.config/am-proxy/config.toml.

    Returns:
        ProxyConfig populated from file, or all-defaults if file missing.
    """
    path = config_path if config_path is not None else DEFAULT_CONFIG_PATH
    if not path.exists():
        return ProxyConfig()

    try:
        with open(path, "rb") as f:
            data: dict[str, Any] = tomllib.load(f)
    except Exception:
        # Malformed TOML — fall back to defaults silently
        return ProxyConfig()

    section: dict[str, Any] = data.get("am_proxy", {})
    return ProxyConfig(
        endpoint=section.get("endpoint", "http://localhost:8000"),
        api_key=section.get("api_key", ""),
        default_project_id=section.get("default_project_id", "default"),
        timeout_seconds=float(section.get("timeout_seconds", 5.0)),
        buffer_ttl_seconds=float(section.get("buffer_ttl_seconds", 300.0)),
        debug=bool(section.get("debug", False)),
    )
```

#### Acceptance Criteria

- [ ] `load_config()` returns `ProxyConfig()` when file does not exist (no exception)
- [ ] `load_config()` reads `[am_proxy]` section from TOML and maps fields correctly
- [ ] Malformed TOML returns defaults silently (no exception)

### Task 3: Implement agents.py — agent name to binary mapping

**File:** `packages/am-proxy/src/am_proxy/agents.py`

**Action:** Create `agents.py` with agent configuration mapping. Full specification:

`AgentConfig` is a simple dataclass:
```python
@dataclasses.dataclass
class AgentConfig:
    binary: str          # Executable name on PATH (e.g. "claude")
    source_agent: str    # Value for source_agent field in ConversationIngestRequest
```

`AGENT_CONFIGS` dict (hardcoded defaults):
```python
AGENT_CONFIGS: dict[str, AgentConfig] = {
    "claude":    AgentConfig(binary="claude",    source_agent="claude_code"),
    "codex":     AgentConfig(binary="codex",     source_agent="openai_codex"),
    "gemini":    AgentConfig(binary="gemini",    source_agent="google_gemini"),
    "opencode":  AgentConfig(binary="opencode",  source_agent="opencode"),
    "kiro":      AgentConfig(binary="kiro",      source_agent="aws_kiro"),
}
```

`get_agent_config()` function:
- Signature: `def get_agent_config(name: str) -> AgentConfig:`
- Looks up `name` (case-insensitive, `.lower()`) in `AGENT_CONFIGS`
- If not found, returns `AgentConfig(binary=name, source_agent=name)` — treats unknown agent name as its own binary name
- Never raises

`detect_installed_agents()` function:
- Signature: `def detect_installed_agents() -> list[str]:`
- Uses `shutil.which(config.binary)` for each entry in `AGENT_CONFIGS`
- Returns list of agent names (keys) whose binary is found on PATH
- Import `shutil` at top of file

Full module must have module-level docstring, Google-style docstrings on all public functions, full type annotations.

#### Acceptance Criteria
- [ ] `AGENT_CONFIGS` contains exactly: claude, codex, gemini, opencode, kiro
- [ ] `get_agent_config("claude")` returns `AgentConfig(binary="claude", source_agent="claude_code")`
- [ ] `get_agent_config("CLAUDE")` returns same result (case-insensitive)
- [ ] `get_agent_config("unknown_agent")` returns `AgentConfig(binary="unknown_agent", source_agent="unknown_agent")` (no raise)
- [ ] `detect_installed_agents()` uses `shutil.which` and returns only agents whose binary exists on PATH
