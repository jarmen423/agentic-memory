  <name>Task 2: background.js — service worker with keepalive, NEW_TURN handler, onInstalled</name>

  <action>
Create background.js implementing:

1. **Keepalive alarm** (must be at top level, not inside a listener):
   ```js
   chrome.alarms.create("keepalive", { periodInMinutes: 0.5 });
   chrome.alarms.onAlarm.addListener(() => {});
   ```

2. **NEW_TURN message handler** — registers via chrome.runtime.onMessage.addListener. Returns false (sync response, no sendResponse needed):
   ```js
   chrome.runtime.onMessage.addListener((msg, sender, sendResponse) => {
     if (msg.type !== "NEW_TURN") return false;
     handle_turn(msg.payload);
     return false;
   });
   ```

3. **handle_turn(payload)** async function — reads chrome.storage.sync, builds ConversationIngestRequest body, fires fetch. Silent failure on ALL paths:
   - Reads: endpoint, api_key, paused, platforms, project_id from chrome.storage.sync
   - Guards: if cfg.paused → return; if !cfg.platforms?.[payload.platform] → return; if !cfg.api_key → return
   - Body matches ConversationIngestRequest exactly:
     ```js
     const body = {
       role: payload.role,
       content: payload.content,
       session_id: payload.session_id,
       project_id: cfg.project_id || "browser",
       turn_index: payload.turn_index,
       source_agent: payload.platform,
       ingestion_mode: "passive",
       source_key: `browser_ext_${payload.platform}`,
     };
     ```
   - fetch call: POST to `${cfg.endpoint || "http://localhost:8000"}/ingest/conversation`, headers Content-Type: application/json and Authorization: Bearer ${cfg.api_key}. Fire-and-forget: `.catch(() => {})`. Entire function wrapped in outer try/catch that also swallows.

4. **onInstalled handler** — opens onboarding page on first install:
   ```js
   chrome.runtime.onInstalled.addListener((details) => {
     if (details.reason === "install") {
       chrome.tabs.create({ url: chrome.runtime.getURL("onboarding/onboarding.html") });
     }
   });
   ```

Use snake_case for JS function and variable names (project convention per CONVENTIONS.md analogy). Private helpers prefixed with `_`. No bare catch clauses — use `.catch(() => {})` for intentional suppression.
