  <action>
    Add a `migrate-temporal` subcommand to `cli.py`.

    **Subparser registration** (add near the existing web-init / chat-init parsers at ~line 1562):
    ```python
    subparsers.add_parser(
        "migrate-temporal",
        help="Backfill temporal fields on all existing relationships (one-time migration)",
    )
    ```

    **Command handler** — add a `_handle_migrate_temporal(args, config)` function that:

    1. Opens a Neo4j session via `ConnectionManager`.
    2. Runs the following Cypher statements in sequence. Each is IDEMPOTENT via
       `WHERE r.valid_from IS NULL`. Group into two categories:

       **Category A — relationships with a Memory parent node (use ingested_at as valid_from):**
       Apply to: `ABOUT`, `MENTIONS`, `BELONGS_TO`, `HAS_CHUNK`, `PART_OF`, `HAS_TURN`, `CITES`.
       Cypher pattern (adapt rel_type per statement):
       ```cypher
       MATCH (m)-[r:ABOUT]->(e)
       WHERE r.valid_from IS NULL
       SET r.valid_from = coalesce(m.ingested_at, "2026-01-01T00:00:00+00:00"),
           r.valid_to = null,
           r.confidence = 0.5,
           r.support_count = 1,
           r.contradiction_count = 0
       ```
       Note: `coalesce(m.ingested_at, "2026-01-01T00:00:00+00:00")` handles Memory nodes that
       predate the `ingested_at` field.

       **Category B — code module relationships (no Memory ancestor, use migration timestamp):**
       Apply to: `DEFINES`, `HAS_METHOD`, `DESCRIBES`, `IMPORTS`, `CALLS`, `PART_OF_PR`.
       Migration timestamp: `"2026-03-25T00:00:00+00:00"` (the Phase 7 migration date).
       Cypher pattern:
       ```cypher
       MATCH ()-[r:DEFINES]->()
       WHERE r.valid_from IS NULL
       SET r.valid_from = "2026-03-25T00:00:00+00:00",
           r.valid_to = null,
           r.confidence = 0.5,
           r.support_count = 1,
           r.contradiction_count = 0
       ```

    3. After each statement, call `result.consume()` and log the counts returned by
       `summary.counters.properties_set` so the user sees progress.

    4. Print a summary: "migrate-temporal: backfill complete. N relationship types processed."

    **CLI dispatch** — in the `main()` function's command dispatch block, add:
    ```python
    elif args.command == "migrate-temporal":
        _handle_migrate_temporal(args, config)
    ```

    **Error handling:** Catch `neo4j.exceptions.ServiceUnavailable` and print a clear message
    ("migrate-temporal: Neo4j unavailable — is Docker running?"). Re-raise on unexpected errors.

    This command is safe to re-run (the WHERE guard ensures already-backfilled rels are skipped).
    Document this in the help string.
