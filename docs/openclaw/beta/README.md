# OpenClaw Private Beta Operations

This directory contains the operator-facing documents for the Phase 15 private
beta program.

The goal of this bundle is narrower than the full GTM plan:

- onboard the first design partners consistently
- define what "active beta user" means in repo-grounded terms
- keep support and feedback collection repeatable while the beta is still small

## Files

- `D:\code\agentic-memory\docs\openclaw\beta\ONBOARDING_RUNBOOK.md`
  - partner intake, install, validation, and first-session checklist
- `D:\code\agentic-memory\docs\openclaw\beta\PARTNER_OPERATIONS.md`
  - how to track the first 5 design partners, feedback cadence, and success
    criteria

## Program Boundaries

This beta assumes:

- self-hosted or operator-managed backend deployment
- npm install surface locked to `agentic-memory-openclaw`
- runtime plugin id locked to `agentic-memory`
- no hosted multi-tenant control plane
- no GA automation promises outside the current release/deployment docs
